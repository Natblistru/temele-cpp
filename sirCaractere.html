Biblioteca cstring

strlen(sir) = lungimea sirului sir
strcpy(sir2, sir1) = copie sir1 în sir2. Returnează true / false (succes / insucces)
strcat(sir1, sir2) = concatenează sir1 + sir2. Rezultatul = în sir1
strcmp(sir1, sir2) = compararea sir1 cu sir2. Returneaza 0 ,dacă sir1 = sir2
strchr(sir1, 'a') = caută simbolul în șir. Returnează: true ,dacă s-a găsit; false ,dacă nu s-a găsit
strstr(sir1, sir2) = caută dacă sir2 se conține in sir1. Returnează: true ,dacă s-a găsit; false ,dacă nu s-a găsit
